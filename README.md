# WeatherAnalyzer

WeatherAnalyzer är ett Pythonprojekt som:
- Läser väderdata från en textfil
- Visar antal dagar
- Hittar högsta temperaturen
- Filtrerar dagar beroende på vädertyp (sunny/rainy/cloudy)
